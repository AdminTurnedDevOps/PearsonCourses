## Go SDK
1. `cd` into `Segment5/advanced/gosdk`
2. `go mod init gosdk`
3. `go mod tidy`
4. `go get helm.sh/helm/v3/pkg/action`
5. `go get helm.sh/helm/v3/pkg/cli`
6. `go run main.go`
